# Employee-Attrition-SQL

Project: Employee Attrition SQL Queries

This project analyzes fictional employee data to understand trends in attrition.

Dataset Fields:
- EmployeeID
- Age
- Department
- Job Role
- Monthly Income
- Education Level
- Attrition Status

Queries Performed:
1. Count employees per department.
2. Calculate attrition rate by department.
3. Compute average income of those who left.
4. Analyze attrition by education level.
5. Determine average age of those who left vs. stayed.

Tools Used:
- SQL (GROUP BY, CASE, AVG, COUNT)
- CSV file as source data for local SQLite or any RDBMS.

This helps HR analysts or decision-makers understand who is leaving and why.
